 URL Shortener App

A simple URL Shortener built using Streamlit and PyShorteners.

## Features
- Shorten long URLs instantly
- Supports TinyURL and Clckru
- URL history
- Clean UI

## Tech Stack
- Python
- Streamlit
- PyShorteners

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
